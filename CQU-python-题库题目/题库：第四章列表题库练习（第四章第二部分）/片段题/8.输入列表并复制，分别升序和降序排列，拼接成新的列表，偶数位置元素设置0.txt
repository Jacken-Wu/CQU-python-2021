【问题描述】
输入一个列表，长度大于2，复制列表，原始列表升序排列，复制列表降序排列。把两个列表拼接成一个新的列表。原始列表在前。最后把偶数位置的元素设置为0，输出新列表。
【输入形式】
按照列表的形式进行输入，包括方括号，元素之间逗号分隔。
【输出形式】
用print直接输出列表
【样例输入】
[3,4,5,6,7,22,2,11,11,22]
【样例输出】
[0, 3, 0, 5, 0, 7, 0, 11, 0, 22, 0, 22, 0, 11, 0, 6, 0, 4, 0, 2]
【样例说明】
样例中合并后的列表为：[2, 3, 4, 5, 6, 7, 11, 11, 22, 22, 22, 22, 11, 11, 7, 6, 5, 4, 3, 2]，新列表的长度是原来的两倍。偶数位置设置0。输出[0, 3, 0, 5, 0, 7, 0, 11, 0, 22, 0, 22, 0, 11, 0, 6, 0, 4, 0, 2]
